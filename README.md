
# AssistantOS

Assistant OS is a open source project to develop an very basic artificial intelligence but very effective for low-level and repetitive tasks.

You can already use

* request parser
    * natural-script
* user management
    * user
    * groups
    * roles
* sheduler
    * reminder


The scope of the AssitantOS is
* home:
* work
* devops
